# منصة إدارة أحداث ومؤتمرات مع جداول وتذاكر ومراسلات
## نظرة عامة
منصة إدارة أحداث ومؤتمرات هي نظام إدارة شاملة يهدف إلى تسهيل عملية تنظيم وتنسيق الأحداث والمؤتمرات. يتيح النظام للمستخدمين إنشاء حسابات، وتسجيل الأحداث، وشراء التذاكر، وإدارة الجداول الزمنية، وإرسال المراسلات.

## هيكل المشروع
هيكل المشروع يتكون من المكونات التالية:
* backend: يحتوي على كود الخادم النصي
* frontend: يحتوي على كود العميل النصي
* database: يحتوي على قاعدة البيانات
* docker: يحتوي على ملفات تكوين الدوكر

## تشغيل البيئة
لتشغيل البيئة، اتبع الخطوات التالية:
1. قم بتثبيت دوكر كومبوز على جهازك.
2. افتح Терمينال وانتقل إلى دليل المشروع.
3. قم بتشغيل الأمر `docker-compose up -d` لتشغيل الخدمات في الخلفية.
4. انتظر حتى يتم تشغيل جميع الخدمات بنجاح.
5. افتح متصفح الويب وانتقل إلى الرابط `http://localhost:8080` لاستخدام المنصة.

## مكونات النظام
النظام يتكون من المكونات التالية:
* **وحدات**:
 + وحدة الأحداث
 + وحدة التذاكر
 + وحدة الجداول الزمنية
 + وحدة المراسلات
* **جداول**:
 + جدول الأحداث
 + جدول التذاكر
 + جدول الجداول الزمنية
 + جدول المراسلات
* **أدوار**:
 + مدير النظام
 + مدير الأحداث
 + مستخدم عادي
